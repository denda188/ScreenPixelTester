在线屏幕检测，测试屏幕是否有像素显示缺陷.
Online screen testing: tests the screen for pixel display defects.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
